# 52 Games Portfolio

Monthly Games의 52개 모바일 게임 포트폴리오입니다.

모든 게임은 `mg_common_game` 공유 라이브러리를 사용하여 개발되었습니다.

---

## 개요

| 항목 | 내용 |
|------|------|
| **총 게임 수** | 52개 |
| **개발 기간** | 2024-2025 |
| **기술 스택** | Flutter, Flame, Dart |
| **대상 플랫폼** | iOS, Android |
| **공유 라이브러리** | mg_common_game |
